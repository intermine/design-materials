InterMine is open source software used to build integrated data warehouses with flexible query interfaces available on the web.

There are InterMine web sites available for fly, rat, yeast, human with zebrafish, mouse and worm coming soon.

Embedded Queries using JavaScript

InterMine provides a web service and a JavaScript API to make it easy to run queries and embed results in other pages. Like this query executed in FlyMine:

You can use the API to create mashups from multiple InterMine sites.

Script APIs

There are also query APIs in Python, Perl and Java for you to write your own scripts and web applications that fetch data from multiple sites.
