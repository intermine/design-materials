Open source data warehouse system from Micklem lab

Using InterMine, you can create databases of biological data accessed by sophisticated web query tools. Parsers are provided for integrating data from many common biological data sources and formats, and there is a framework for adding your own data. InterMine includes an attractive, user-friendly web interface that works 'out of the box' and can be easily customised for your specific needs, as well as a powerful, scriptable web-service API to allow programmatic access to your data.

Getting started

A sophisticated web application provides query access to users. The interface allows creation of custom queries, includes template queries - web forms to run 'canned' queries and can upload and operate on lists of data. It is possible to create and add widgets to analyse lists with graphs and enrichment statistics. An important feature is that an admin user can publish new template queries, change report pages and create public lists at any time without any programming. InterMine instances offer a range of services through a RESTful HTTP interface. These services expose data and operations that are accessible from any script or programme that can communicate over HTTP. To facilitate this, we also publish client libraries in a number of popular languages to simplify the construction of webservice requests, and the parsing of webservice responses.
