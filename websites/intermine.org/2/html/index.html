Open source data warehouse system for biological data integration

InterMine is a data warehouse system built for the integration and analysis of complex biological data. Developed by the Micklem lab at the University of Cambridge, InterMine enables the creation of biological databases accessed by sophisticated web query tools. Parsers are provided for integrating data from many common biological data sources and formats, and there is a framework for adding your own data. InterMine includes an attractive, user-friendly web interface that works 'out of the box' and can be easily customised for your specific needs, as well as a powerful, scriptable web-service API to allow programmatic access to your data.

Complex data integration

InterMine was developed with the complexity of biological data in mind. The data model is flexible and extensible, and a range of data parsers is provided to facilitate the data loading. A sophisticated identifier resolution system updates all identifiers to the most current version using a priority system, and multiple post-processing checks ensure the consistency of the data integration.

Learn More

Fast and flexible querying

Complex queries can be constructed flexibly to mine across the integrated datasets, enabling researchers to answer sophisticated biological questions. The query optimisation method is constructed around the use of precomputed tables, meaning that the data schema does not need to be denormalized to optimise query speed. A user's query workflow can also be automated using InterMine web services.

Pre-constructed web application

The InterMine web application presents a graphical interface suitable for both first time and advanced users. Information can be retrieved for individual objects such as genes, analysis can be performed on lists of genes to run a range of analytics on it, and queries can be constructed using a query builder, or run from our collection of template queries. Users also have a private myMine workspace, where the results of their queries are stored.

Share your lists with other users from myMine, GoogleDocs style! Now in testing phase at http://beta.flymine.org - let us know what you think.
